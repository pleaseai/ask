Build a toolbar with grouped buttons using Nuxt UI components.

Requirements:

1. Create a toolbar with 3 button groups:
   - **Text formatting group**: Bold, Italic, Underline buttons with appropriate Lucide icons
   - **Alignment group**: Left, Center, Right align buttons with icons
   - **Actions group**: A "Save" button with label + a dropdown chevron button
2. Group each set of related buttons together so they appear visually connected (no gaps between buttons in the same group)
3. Use `variant="outline"` and `color="neutral"` for all buttons
4. The groups should support different sizes — use `size="sm"` for the formatting group and default size for others
5. Add a vertical orientation group as well (e.g., a sidebar with stacked navigation buttons)

Focus on correct Nuxt UI v4 component usage for grouping buttons together.
